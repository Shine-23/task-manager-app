# TASK MANAGER APP

A **Task Management system** built with:
- Spring Boot (Backend)
- ReactJS (Frontend)
- MySQL Database

## Repository Structure
`/backend` => Spring Boot backend

`/frontend` => ReactJS frontend

## Prerequisites
- Java 17+
- Maven
- Node.js
- NPM
- MySQL

## Installation Steps
### Backend Setup
1. Configure MySQL in `/backend/src/main/resources/application.properties`.
2. Run:
```bash
cd backend
./mvnw clean install
./mvnw spring-boot:run
```
